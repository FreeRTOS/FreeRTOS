Contains projects that demonstrate the OTA library.

Over-the-air (OTA) updates allow you to deploy firmware updates to one or more
devices in your fleet. Although OTA updates were designed to update device
firmware, you can use them to send any files to one or more devices registered
with AWS IoT.
https://docs.aws.amazon.com/freertos/latest/userguide/freertos-ota-dev.html
